# Projeto de Extensão: Site-Vitrine "Caminhos do Sabores"

![Caminhos do Sabores Homepage](iPad.png)

## 1. Visão Geral

Este projeto é um site-vitrine estático e responsivo desenvolvido para a "Caminhos do Sabores", um empreendimento local de doces gourmet. O objetivo principal do site é servir como um catálogo digital profissional, otimizando o processo de vendas da empreendedora, que antes era 100% dependente do WhatsApp para exibir produtos e anotar pedidos.

## 2. Contexto Acadêmico

Este site foi desenvolvido como atividade prática para o componente curricular **PROJETO DE EXTENSÃO II - DESENVOLVIMENTO WEB**, do Curso Superior de Tecnologia em Desenvolvimento Web, da Faculdade Anhanguera.

O projeto está alinhado ao **PROGRAMA DE INOVAÇÃO E EMPREENDEDORISMO**, cuja finalidade é aplicar os conhecimentos técnicos do curso para desenvolver "soluções computacionais que oportunize o desenvolvimento comercial e econômico da região" da zona leste de São Paulo/SP.

* **Aluna:** Thaís Vieira Silva 
* **Parceiro:** Caminhos do Sabores (Empreendimento local)

## 3. Funcionalidades (Features)

O site é composto por três páginas principais, todas com design responsivo (mobile-first):

* **Home (`index.html`):** Página principal com destaques dos produtos e chamada para ação (CTA) para o WhatsApp.
* **Cardápio (`cardapio.html`):** Um menu interativo no formato "Accordion" (Sanfona), onde os clientes podem expandir categorias (Brigadeiros, Bolos, Trufas) para ver a lista de itens e seus respectivos preços.
* **Contato (`contato.html`):** Informações sobre o processo de encomenda (prazos, pagamento) e um botão direto para o WhatsApp.

## 4. Tecnologias Utilizadas

Este projeto foi construído "do zero" utilizando as seguintes tecnologias:

* **HTML5:** Para a estrutura semântica do site.
* **CSS3:** Para toda a estilização personalizada (cores, fontes, etc.).
* **Bootstrap 5:** Utilizado como framework base para o grid responsivo, componentes (Navbar, Cards, Accordion) e otimização de tempo.
* **Google Fonts:** Para as fontes personalizadas ('Pacifico' para títulos e 'Montserrat' para o corpo).
* **Visual Studio Code:** Editor de código utilizado no desenvolvimento.

## 5. Como Executar o Projeto

Este é um projeto estático e não requer um servidor para ser executado.

1.  Clone este repositório (ou baixe os arquivos ZIP).
2.  Navegue até a pasta do projeto (ex: `SITE-VITRINE`).
3.  Abra o arquivo `index.html` diretamente no seu navegador de preferência.
